# LOFAVER Eyelash Storefront

Pure English static storefront for the LOFAVER brand.

## Files

- `index.html`: landing page
- `styles.css`: site styles
- `script.js`: light interaction for mobile menu and reveal animations
- `brand-logo.png`: storefront logo asset
- `support-card.jpg`: after-sales support card

## Local Preview

Run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
